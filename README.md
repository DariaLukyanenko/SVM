# SVM

# SVM: классификация биомедицинских данных

## Описание
Блокнот посвящён методу опорных векторов (**SVM**) и его применению в задаче бинарной классификации.  
Материал сочетает краткую теорию, визуализацию геометрического смысла SVM и практический пример на медицинском датасете.

## Что есть в блокноте
- постановка задачи бинарной классификации;
- разделяющая гиперплоскость и правило предсказания;
- опорные векторы и максимальный зазор;
- мягкий зазор и роль гиперпараметра `C`;
- двойственная задача и матрица Грама;
- ядровой трюк;
- сравнение линейного, полиномиального и RBF-ядер;
- влияние параметра `gamma`;
- связь зазора с обобщающей способностью;
- практическое обучение SVM на реальном биомедицинском датасете.

## Данные
В практической части используется **Wisconsin Breast Cancer Dataset**.

Характеристики:
- **569 образцов**;
- **30 числовых признаков**;
- задача бинарной классификации:
  - `0` — злокачественная опухоль;
  - `1` — доброкачественная опухоль.

Распределение классов:
- злокачественные: **212**;
- доброкачественные: **357**.

## Что сделано в блокноте
- загрузка и первичный анализ данных;
- визуализация пар признаков;
- проекция данных в 2D через **PCA**;
- разделение на train/test;
- стандартизация признаков;
- обучение линейного SVM;
- обучение SVM с RBF-ядром;
- сравнение моделей по accuracy;
- построение confusion matrix;
- подбор гиперпараметров `C` и `gamma` через `GridSearchCV`;
- оценка лучшей модели через classification report.

## Модели

### 1. Линейный SVM
Используется как базовая модель для линейно разделимых или почти линейно разделимых данных.

### 2. SVM с RBF-ядром
Позволяет строить нелинейную границу и лучше работать со сложной структурой данных.

### 3. SVM с подбором гиперпараметров
Для RBF-модели выполняется поиск лучших параметров:
- `C`;
- `gamma`.

## Основные результаты

### Без подбора гиперпараметров
- **Линейный SVM**
  - train accuracy: **0.9912**
  - test accuracy: **0.9737**

- **RBF SVM**
  - train accuracy: **0.9934**
  - test accuracy: **0.9737**

### После GridSearchCV
- лучшие параметры: **C = 10**, **gamma = 0.01**
- лучшая CV accuracy: **0.9802**
- test accuracy: **0.9825**

## Ключевые выводы
- SVM хорошо подходит для задач бинарной классификации на табличных биомедицинских данных.
- Линейный SVM уже даёт высокое качество на этом датасете.
- RBF-ядро становится полезным, когда граница между классами нелинейна.
- Подбор `C` и `gamma` улучшает итоговое качество модели.
- Основная идея SVM — искать разделяющую границу с максимальным зазором, а решение определяется опорными векторами.
